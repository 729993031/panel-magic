# Panel-Magic
🌈🐴🐂🐱🐶🐷🌈

Panel-Magic 是基于 Angular8 实现的在线可视化编辑应用，如果你习惯使用 PS，那么它的体验会给你带来熟悉的感觉

![demo](/assets/panel-magic-demo.png)

## 前置要求

需要安装 `@angular/cli` 的最新版本，并安装了 `Angular 8` 版本，
然后直接在命令行输入
```base
yarn run start
```
即可运行调试

## 开发
```bash
$ git clone https://github.com/Ricbet/panel-magic.git
$ cd panel-magic
$ yarn install
$ yarn run start
```
